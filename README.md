# Chat.IA - Versão Suprema

A Chat.IA é uma inteligência artificial autônoma projetada para ser a sua assistente definitiva no mundo dos games e desenvolvimento. Ela combina a potência do modelo Gemini 1.5 Pro do Google com recursos de automação de desktop, permitindo que ela não apenas responda às suas perguntas, mas também interaja diretamente com o seu computador.

## Recursos Principais

- **Chat Inteligente:** Respostas claras, precisas e profissionais sobre desenvolvimento de jogos (focada em Roblox e Lua) e tópicos gerais.
- **Automação de Desktop:** Capaz de digitar códigos diretamente no Roblox Studio, economizando seu tempo.
- **Histórico de Chat:** Salva suas conversas para consulta futura.
- **Interface Profissional:** Uma interface de chat moderna e fácil de usar, construída com Streamlit.
